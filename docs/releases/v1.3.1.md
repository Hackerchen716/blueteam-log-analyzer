# BlueTeam Log Analyzer v1.3.1

v1.3.1 新增 RDP/登录专项分析模式，面向 Windows Security 日志中的 4624/4625 登录事件。该能力参考并吸收了 duckpigdog 在 PR #16 中提出的方向。

## 新增

- 新增 `--rdp` 参数：仅保留带远程来源 IP 的 Windows 4624/4625 登录事件，适合大日志中快速筛出 RDP/网络登录线索。
- 终端专项分析新增登录明细：展示账号、来源 IP、登录类型、工作站、认证包、失败原因和状态码。
- Remote Workspace 中的 `bla FILE --rdp` 同步支持 RDP/登录专项分析，远程主机仍然不需要安装 Python、pip 或 BLA。
- 新增 `sample_logs/windows_rdp_sample.xml`，用于验证 RDP 专项过滤和终端输出。

## 修复

- `--rdp` 过滤只发生在专项模式中，通用 Windows XML/EVTX 解析器保持完整解析，不会全局丢弃无来源 IP 的 4624 事件。
