# BlueTeam Log Analyzer v1.2.0 Release Notes

v1.2.0 是一次架构和扩展能力更新。本次更新重点不是继续增加检测规则，而是稳定后续功能迭代需要的核心接口。

## 主要更新

- 新增 `ParserRegistry`，新的日志源可以通过注册方式接入。
- 新增 `DetectorRegistry`，新的检测器可以按 profile 组合。
- 新增 `parse_content()`，支持内存日志内容进入现有解析流程。
- CLI 新增 `--type`，自动识别不准确时可以强制指定解析器。
- Pipeline 避免重复 enrichment，分析链路更清晰。
- 新增 `docs/architecture.md`，记录后续扩展的架构边界。
- README 更新为 v1.2.0，并替换新的项目 banner。

## 兼容性

- 继续支持现有 `bla <path>` 用法。
- 继续支持 `validate-rules`、`benchmark`、`explain` 子命令。
- 继续支持 HTML、JSON、CSV、IOC、SARIF 和 `--out` 标准报告目录。

## 后续计划

- v1.3：Remote Collector MVP，支持只读 SSH 日志采集。
- v1.4：Host Triage MVP，输出结构化主机排查报告。
- v1.5：Parser ecosystem 和 P0 adapter 拆分。
- v2.0：轻量离线 IR Workbench。
