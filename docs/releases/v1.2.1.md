# BlueTeam Log Analyzer v1.2.1 Release Notes

v1.2.1 是一次面向实战输出可信度的小版本更新，重点修正 EVTX 依赖缺失时的误导性体验，并降低 Windows Security 初始化事件造成的高危噪音。

## 修复与增强

- EVTX 二进制日志缺少 `python-evtx` 时，解析流程会明确失败并提示安装或 `wevtutil` 转 XML，不再生成看似正常但没有真实 EVTX 事件的报告。
- Windows 账户管理事件补充操作者、目标账户、目标组、成员 SID、证据强度和误报提示，减少 `- -> Users`、`- -> None` 这类字段不完整输出。
- `WDAGUtilityAccount` 等系统初始化账户创建不再作为持久化高危告警。
- `Users`、`IIS_IUSRS`、`None` 等低风险/默认组成员变更不再触发“账户添加到特权组”告警。
- `Administrators`、`Remote Desktop Users`、`Remote Management Users`、`Backup Operators` 等敏感组仍保留高危特权组告警。
- 本机 `localhost` 或机器账号导致的 4648 显式凭据事件不再直接定性为 Pass-the-Hash 横向移动；远程显式凭据异常保留为需结合网络、RDP、SMB、EDR 证据确认的告警。
