# BlueTeam Log Analyzer v1.3.8

发布项如下：

## 修复与增强

- 终端输出的长中文证据、路径和命令行改为按显示宽度截断，避免 CJK 字符把拓扑树撑到换行错位。
- README 和发布说明收敛公开表述，保留功能变化、验证范围和交付信息，不再展开描述终端标题区的具体文案。
- README 的终端示例聚焦分析结果和攻击拓扑，减少与真实运行环境无关的静态开头文本。

## 验证重点

- 显示宽度计算继续使用 East Asian Width，中文宽字符按两列处理。
- 本次优化只影响终端展示和文档，不改变 JSON、CSV、SARIF、manifest、规则或解析结果结构。
- 发布前继续运行 `python3 scripts/release_check.py --build` 和 PyPI clean-venv smoke。
